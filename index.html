Installation

This is a quick start tutorial for using yo dnn. For a more in depth setup take a look at the code lab.

First, install Yeoman and generator-dnn using npm (we assume you have pre-installed node.js).

npm install -g yo
npm install -g generator-dnn

You will also want to install gulp.js

npm install -g gulp

Then generate your new project:

mkdir my-project-name
cd my-project-name
yo dnn

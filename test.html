What is generator-dnn?

generator-dnn helps you kickoff your DNN/Evoq Module or Theme development.

You can scaffold MVC, SPA, or Persona Bar modules. You can also scaffold the start of a Theme.

Installation

First, install Yeoman and generator-dnn using npm (we assume you have pre-installed node.js).

npm install -g yo
npm install -g generator-dnn

Then generate your new project:

mkdir my-project-name
cd my-project-name
yo dnn

Troubleshooting

Most issues can be found by running

yo doctor

The doctor command will diagnose and provide steps to resolve the most common issues.
